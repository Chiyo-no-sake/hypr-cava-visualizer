# Contributing

Thanks for your interest in improving hypr-cava-visualizer!

## Requirements

- Python 3.10+
- GTK4
- gtk4-layer-shell
- cava
- A running Hyprland session for testing

## Workflow

1. Fork the repo and create a feature branch from `main`.
2. Make your changes. Keep it simple (KISS) -- small, focused PRs are preferred.
3. Test manually on Hyprland before submitting. There is no automated test suite, so verify that the visualizer renders correctly and responds to config changes.
4. Open a pull request against `main` with a clear description of what changed and why.

## Guidelines

- One logical change per PR.
- Follow the existing code style.
- If you add a dependency, justify it in the PR description.
- Bug fixes and UX improvements are always welcome.
